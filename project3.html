My Travel Map App

HTML, Javascript, React.js, Leaflet

What I made?

On my spare time I started working on a travel mapping app. Ever since I went to Japan in 2018, I had kept this in project in a corner of my mind despite not knowing anything about coding then. I have just barely scratch the surface on this project, but everything on here is a learning curve.

How I made it?

So far I have only added some specifications for the map rendered when the page loads. I have also added a few markers with limited interactivity. Eventually i would like this application to be able to link to Instagram and load pictures taken throughout my journey, using the geolocation tags.
