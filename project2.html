Google Books API Search Engine

HTML, CSS/SCSS, JavaScript, React.JS

What I made?

After a couple of weeks of intense Javascript learning and an introduction to APIs, we were tasked on created a search engine using the Google Books public resources.

How I made it?

Using React.JS, I created resusable components handling different parts of the logic necessary for this application to work. Learning about Axios, and having to search beyond the scope of the course to be able to complete this project, was very rewarding.

What I Learned?

After finishing this small app, I was much more comfortable with handling API calls and responses. Learning about, and how to use Postman was certainly very useful too.
