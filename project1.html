My First Project

Calculator

HTML, CSS/SCSS, Bootstrap

What I made?

This was the first project we had to deliver during our 12-week Software Engineer Course. A simple, fully functioning calculator, capable of handling basic math operations.

How I made it?

Starting with writing down the logic for being able to handle a simple operation, I then expended on having multiple operations consecutively. Breaking down the logic behind each button was key to making sure that this application could handle most use cases.

Using what I learned so far about for loops and conditional logic, I was able to flesh out a functionning prototype. The styling waas handled with CSS/SCSS, making use of the Bootstrap library.

What I Learned?

This being our very first project, I realised how important it was to take a step back and being able to break down the logic behind your target code. I found that the more granular I went, the easiest it was later on.
